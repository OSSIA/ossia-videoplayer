# ossia-videoplayer

An OSCQuery-enabled video player.

# Dependencies

- libossia
- libmpv

# Usage

    # Starts a player
    ./ossia-videoplayer
    
    # Starts a player with 3 videos
    ./ossia-videoplayer -n 3
